Self Soul - 类人AGI人工智能系统

注意：这是一个系统概览页面，由于缺少Node.js环境，无法运行完整的Vue应用。如果需要完整体验，请安装Node.js并在app目录下运行npm run dev。

系统架构

前端技术栈

  • Vue 3 + Vite - 现代化前端框架
  • Vue Router 4 - 路由管理
  • Pinia - 状态管理
  • WebSocket - 实时通信

后端技术栈

  • FastAPI - 高性能Web框架
  • UnifiedManagerModel - 核心模型管理
  • 11个专业子模型 - 覆盖不同领域功能
  • 内存优化系统 - 智能内存管理

核心功能模块

1. 聊天系统

文件位置：app/src/views/ChatFromScratch.vue

功能：

  • 智能对话交互
  • 多轮对话管理
  • 上下文理解
  • 支持多模态输入

2. 知识管理

文件位置：app/src/views/KnowledgeView.vue

功能：

  • 知识导入与管理
  • 知识库查询
  • 知识图谱构建
  • 知识更新与维护

3. 系统监控

文件位置：app/src/views/MonitorDashboard.vue

功能：

  • 实时内存监控
  • CPU使用率跟踪
  • 模型状态监控
  • 系统负载分析

4. 训练控制台

文件位置：app/src/views/TrainingControlPanel.vue

功能：

  • 模型训练管理
  • 参数调优
  • 训练进度跟踪
  • 训练结果分析

5. 情感识别

文件位置：app/src/models/EmotionModel.js

功能：

  • 情感分析
  • 情绪识别
  • 情感反馈
  • 情感记忆

后端API测试

虽然前端无法完全运行，但后端服务已在端口8000和8001启动。您可以使用Python脚本测试API功能：

# 测试聊天API import requests url = "http://localhost:8001/api/models/8001/chat" headers = { "Content-Type": "application/json" } data = { "message": "四大发明是什么？", "session_id": "test_session_123" } response = requests.post(url, json=data) print(response.json())

启动指南

  1. 安装Node.js：https://nodejs.org/
  2. 打开命令行，进入app目录
  3. 运行 npm install 安装依赖
  4. 运行 npm run dev 启动开发服务器
  5. 访问 http://localhost:5173
当前HTTP服务器：已在端口5000启动，您可以通过以下地址访问：
  • 系统概览：http://localhost:5000/system_overview.html
  • 原始HTML：http://localhost:5000/index.html（功能受限）
