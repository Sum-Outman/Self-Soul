# 贡献者公约行为准则

## 📜 我们的承诺

我们作为成员、贡献者和领导者承诺，使参与我们社区的所有人免于骚扰，无论其年龄、体型、可见或不可见的残疾、种族、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性取向如何。

我们承诺以有助于建立开放、友好、多样化、包容和健康社区的方式行事和互动。

## 🎯 我们的标准

有助于为我们的社区创造积极环境的行为示例包括：

* 表现出对他人的同理心和善意
* 尊重不同的意见、观点和经验
* 建设性地给予和接受反馈
* 承担责任并向受我们错误影响的人道歉，并从经验中学习
* 专注于不仅对我们个人有利，而且对整个社区有利的事情

不可接受的行为示例包括：

* 使用性暗示的语言或图像，以及任何形式的性关注或接触
* 挑衅、侮辱或贬损性评论，以及个人或政治攻击
* 公开或私下骚扰
* 未经明确许可发布他人的私人信息，例如实际地址或电子邮件地址
* 其他在专业环境中合理认为不适当的行为

## 🛡️ 执行责任

社区领导者负责澄清和执行我们的行为标准，并将对任何他们认定为不适当、威胁、冒犯或有害的行为采取适当和公平的纠正措施。

社区领导者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、维基编辑、问题和其他贡献，并将在适当时沟通执行措施的原因。

## 🌍 适用范围

本行为准则适用于所有社区空间，并且当个人在公共空间正式代表社区时也适用。代表我们社区的示例包括使用官方电子邮件地址、通过官方社交媒体账户发布或在线或离线活动中担任指定代表。

## ⚖️ 执行

虐待、骚扰或其他不可接受的行为可以向负责执行的社区领导者举报。所有投诉都将得到及时和公平的审查和调查。

所有社区领导者都有义务尊重举报事件的安全性和隐私性。

## 📋 执行指南

社区领导者将遵循以下社区影响指南来明确违反本准则的后果：

### 1. 纠正
**社区影响**：使用不恰当的语言或其他在社区中被视为不专业或不受欢迎的行为。

**后果**：社区领导者将提供私下的书面警告，明确说明违规行为的性质，并解释为何该行为不适当。可能会要求公开道歉。

### 2. 警告
**社区影响**：通过单个事件或一系列行为造成的违规。

**后果**：警告并说明持续行为的后果。在指定时间段内不得与相关人员互动，包括主动与行为准则执行者互动。这包括避免在社区空间以及外部渠道如社交媒体中互动。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁
**社区影响**：严重违反社区标准，包括持续的不当行为。

**后果**：在指定时间段内暂时禁止与社区的互动。在此期间，不得与相关人员进行公开或私下互动，包括与行为准则执行者的互动。违反这些条款可能导致永久封禁。

### 4. 永久封禁
**社区影响**：表现出违反社区标准的模式，包括持续的不当行为、骚扰个人或攻击或贬低某个群体的成员。

**后果**：永久禁止在社区内的任何公开互动。

## 📞 举报

如果您遇到或目睹违反本行为准则的行为，请通过以下方式举报：

**项目负责人邮箱**: [负责人邮箱]
**GitHub Issue**: [创建Issue](https://github.com/your-org/self-soul-b/issues/new?template=behavior_report.md)

所有举报将由至少两名社区领导者审查，他们将遵循上述执行指南确定适当的措施。我们将尊重您报告的保密性，以保护您的安全。
